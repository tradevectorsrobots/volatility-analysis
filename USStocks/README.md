# US Stocks Historical Data

This folder contains historical price data for US stocks in CSV format.

## Data Format

Each CSV file contains:
- Date
- Open
- High
- Low
- Close
- Volume
- Adjusted Close (if applicable)

## Usage

Upload your historical stock data CSV files to this folder for volatility analysis.

## File Naming Convention

Recommended format: `TICKER_start_date_end_date.csv`

Example: `AAPL_2020-01-01_2026-03-10.csv`
